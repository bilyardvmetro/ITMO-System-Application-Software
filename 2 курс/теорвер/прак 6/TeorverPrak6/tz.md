---
вариационный ряд = сортануть выборку по возрастанию

макс для него

мин для него

размах = разница между max и min

статистический ряд = список пар (элемент, его количество в выборке)

эмпирическая функция распределения по этому ряду (```F(x) = (кол-во значений < x) / (объем выборки)```)

частоты = количества элементов в выборке

частотности = отношение частот к количеству элементов выборки

длина интервала = формула стерджеса = `h= (x max - x min) / (1+log2(n))`

количество интервалов = `1+log2(n)`

начало интервала = `x min - h/2`

полигон частотностей = график из ломаных, соединяющих точки `(xi, pi = частотность)`

гистограмма частотностей = график из прямоугольничков с основанием `h` и высотой `pi/h`
