Как пользоваться

  1. Нажмите кнопку "Выберите файл": Затем выберите картинку, которую хотите загрузить.
  2. Настройте ползунки яркости, контраста и блюра по вкусу. Для применения изменений нажмите соответствующую кнопку. Каждый раз, меняя ползунок, для обновления картинки нажмите кнопку.
  3. Для получения черно-белого изображения нажмите кнопку "Черно-белое".
